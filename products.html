Hand-Off Engine (가출원 완료 · KR Provisional Patent)

Hand-Off Engine은 여러 번에 걸친 AI 세션·대화·작업 흐름을 하나의 연속된 맥락으로 관리하기 위한 엔진 구조입니다.

The Hand-Off Engine is designed to manage multiple AI sessions, conversations, and task flows as a single continuous context.

  • 세션 간 작업 상태를 구조화된 형태(YAML 등)로 보존
  • 다음 세션에서 안전하게 불러와 이어서 작업 가능
  • 대화 내용이 아닌, “작업 맥락”을 중심으로 저장

이 기술은 현재 대한민국 특허청에 가출원(임시출원)된 상태입니다. 정식 심사 및 해외 출원 전략에 따라 세부 구조는 순차적으로 공개될 예정입니다.

  • 가출원 번호: 10-2025-0167794
  • 출원일: 2025-11-08

A provisional patent application has been filed with the Korean Intellectual Property Office (Application No. 10-2025-0167794, Filing Date: 2025-11-08). Detailed algorithms and implementation remain confidential while the application is pending.

Junior Language (자연어 코딩 연구 프로젝트)

Junior Language는 자연어 명령만으로 프로그램을 구성하는 자연어 기반 프로그래밍 언어를 목표로 하는 연구 프로젝트입니다.

Junior Language is a research project aiming to build a natural-language-based programming language where programs are constructed purely from human-readable commands.

  • 사람이 말하듯 명령하면 코드 스켈레톤을 생성하는 개념
  • 에러 대신 “수정 제안”을 반환하는 안전한 구조를 지향
  • 다국어 지원 및 EF-1 언어 시스템과의 연동을 장기 목표로 함

이 프로젝트는 현재 연구/실험 단계이며, 상용 프로그램이나 서비스로 제공되지 않습니다. 아직 별도의 특허 출원도 진행하지 않았기 때문에, 구체적인 구현 방법, 내부 데이터 구조, 알고리즘 등은 외부에 공개하지 않습니다.

The project is currently at a research/experimental stage and is not provided as a commercial program or service. No patent application has been filed yet, so implementation details and internal data structures remain undisclosed.
